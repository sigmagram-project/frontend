# Image Board — Frontend

Talks to the backend at
`http://localhost:8080` (see `src/api/client.js`).

## Run it

```
npm install
npm run dev
```
Runs on http://localhost:5173. Requires the backend to be running for anything to actually work.

Run frontend tests with:
```
npm test
```

## Design team

`src/components/Logo.jsx` is the placeholder for the product logo/branding.
`src/index.css` is intentionally minimal/unbranded — restyle freely.
